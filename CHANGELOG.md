# Changelog

All notable changes to this project will be documented here.

## [v0.1.0] - 2026-02-16

- Initial public release: central clock + layered HUD, celestial volvelle ring, modular widgets (weather, now playing, net/sys, sitrep), theme.lua + theme-sitrep.lua, scripts/fonts, screenshots, MIT license.

## [v0.2.0] - 2026-02-21

- Added layout scaling + draw scaling controls (manual/auto), plus center-anchored positioning and embedded corner widgets.
- Added screen-size helper + startup integration and expanded doctor diagnostics around layout inputs.
- Fixed scaling across weather (vline + forecast), corner widget outer rings, SITREP panel, and notes/music/doctor panels.
